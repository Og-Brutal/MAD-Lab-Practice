# Firebase Authentication Checklist

Use this checklist when reviewing or running the `FireBaseAuthetication` project.

- [ ] Verify `google-services.json` is present in `FireBaseAuthetication/app/`
- [ ] Confirm Firebase dependencies are configured in `app/build.gradle.kts`
- [ ] Run the app and test email sign-up
- [ ] Test login flow and access to the home activity
- [ ] Ensure no private keys or credentials are committed to git
